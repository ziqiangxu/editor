# Git静态网页编辑器 Linux客户端 
应用简介
- 可以进行文本的编辑和字符的查找
- 支持拖动文件进行编辑
- 在设置好git目录之后对过的文件快捷提交
> 本应用目标用户是使用类似gitpage的用户，还可以在应用启动时默认打开指定文件（推荐设置为模板文件，然后另存发布）

安装方法：
```
cd install
./install
```
注：请不要使用sudo ./install，否则会安装出错
